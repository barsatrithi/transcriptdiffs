# transcriptdiffs

This was a workflow I did a while back.

Basically, say you have a conversation with a AI voice assistant through VAPI(or any other voice assistant phone call). 
Fix the file and make any edits that the AI did not pick up on. Then you run the script and it will find the transcript pairs of the before and aftert file and generate the DIFFs for the transcript and the JSON file, and  provide evaluation metrics.

Before running the DIFFs for the JSON, there is also a pretty print script for this workflow if the JSONs is not structured nicely.

I am not too sure how helpful this will be for others, because this script was very specific to a workflow before, but just putting it here for reference.
